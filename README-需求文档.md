# Image Background Remover - 产品需求文档

> 面向国外小电商卖家的AI抠图工具

---

## 1. 产品概述

### 1.1 产品定位

| 字段 | 内容 |
|------|------|
| **产品名称** | Image Background Remover |
| **产品定位** | 面向国外小电商卖家的快速抠图工具 |
| **核心价值** | 低价 + 批量处理 + 简单易用 |
| **目标平台** | Web网站（移动端友好） |

### 1.2 核心功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 单图抠图 | 上传图片 → AI自动抠图 → 下载 | P0 |
| 批量抠图 | 一次上传多张，批量处理 | P1 |
| 自定义背景 | 抠图后换背景色/纯色/图片 | P2 |

### 1.3 目标用户

| 用户画像 | 描述 |
|----------|------|
| **主要用户** | Etsy、Shopify、Amazon FBA小卖家 |
| **用户痛点** | 产品图需要白底图，手动PS太慢 |
| **付费意愿** | $3-15/月可接受 |

---

## 2. 功能详情

### 2.1 核心用户流程

```
用户访问网站
    ↓
上传图片（拖拽或点击）
    ↓
AI自动处理（显示进度）
    ↓
预览结果 + 下载
    ↓
（可选）换背景色
```

### 2.2 页面结构

| 页面 | 功能 |
|------|------|
| 首页 | 上传区 + 结果展示 + 下载 |
| - | - |

### 2.3 功能规格

#### 单图处理
- 支持格式：JPG、PNG、WebP
- 最大文件大小：10MB
- 处理时间：3-10秒（取决于图片大小）
- 输出格式：PNG（透明背景）

#### 批量处理（P1）
- 同时处理数量：最多10张
- 显示处理进度
- 支持批量下载

#### 背景替换（P2）
- 纯色背景：白、黑、透明
- 自定义颜色选择器

## 3. 技术方案

### 3.1 技术栈

| 组件 | 方案 | 说明 |
|------|------|------|
| 前端框架 | React + Vite | 快速开发，热更新 |
| 样式 | Tailwind CSS | 原子化CSS，快速样式 |
| 后端 | Cloudflare Worker | 边缘计算，免费额度 |
| AI服务 | Remove.bg API | $0.09/张 |
| 存储 | 内存处理 | 不存储图片，保护隐私 |
| 部署 | Cloudflare Pages | 免费CDN |

### 3.2 系统架构

```
┌─────────────┐
│   用户浏览器  │
└──────┬──────┘
       │
       ▼
┌─────────────┐
│ Cloudflare  │  ← 前端（Pages）
│   Pages     │
└──────┬──────┘
       │ /api
       ▼
┌─────────────┐
│  Cloudflare │  ← 后端（Worker）
│   Worker    │
└──────┬──────┘
       │ v1.0/removebg
       ▼
┌─────────────┐
│  Remove.bg  │  ← AI抠图服务
│    API      │
└─────────────┘
```

### 3.3 数据流

```
1. 用户上传图片 → 前端表单
2. 前端 → POST /api
3. Worker获取图片 → 转发给Remove.bg
4. Remove.bg返回透明背景图
5. Worker返回给用户
6. 全程不存储图片（内存处理）
```

### 3.4 环境变量

| 变量名 | 说明 | 必填 |
|--------|------|------|
| `REMOVE_BG_API_KEY` | Remove.bg API Key | 是 |
